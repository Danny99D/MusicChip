PNG & PSD Chip Files

Create a Excel File

You'll need a Excel File, the Row 1 need to have "name" and "steam", "youtube" is optional but is nice to have the link if needed.
Lower Case for name, steam and youtube, styles not needed.

You can name the Pages and select different pages.

Upload the Chip Image and Tracks to Steam Cloud

In the Tabletop (TTS) Menu, in Modding > Cloud Manager you can upload up to 100GB for free in Steam.

Upload here the Images for the Chip and the Music Tracks, I recommend organize the files in folders and use MP3 for the Music files.

Convert the Excel File to a Json File

Before convert the File, copy the Steam Links of the Images and Music from the Steam Cloud and paste in the Excel in the "steam" column, you can copy the Link just clicking it in the Cloud Manager.

I use this Page to convert the Excel. You don't need to download the Json File, just copy it.

Fill the Form

First You'll need to paste the JSON code from the Converted Excel to the "JSON Code here" field.

After pasting the JSON, the Select Box above will have the Name of the Pages of the Excel, select one of the Pages and the "Track Preview" will have the list of Tracks and a preview from the Steam Cloud Link.

Name, Description and Images are self explanatory, You can leave the Image Link Empty and the Chip will use the Default ones if just want to test it.

Create the Chip JSON

After pasting the JSON, the "Convert Menu" Button will be available. You need to select a Page, write a Name and a Description to convert it.

Click "Copy to Clipboard" or select all the Text with Ctrl + A to copy it.

Go to your Saved Objects Folder wich is located in "Documents/My Games/Tabletop Simulator/Saves/Saved Objects/"

Create a file.json (In my case is test.json) with the notepad and paste here the Json Code.

The Saved Object updates itself in Tabletop when you save the File, so you can use the same file to create various Chips without need to create a new file.

Save the Json, draw the Saved Object without Image preview and Save the Chip to let Tabletop create the Preview Image.
